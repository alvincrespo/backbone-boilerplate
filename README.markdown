# Backbone.js Boilerplate

## Purpose

The purpose of this boilerplate is to ease in frontend devs into Backbone.js development by providing a framework of organization to a frontend mvc application.

## Tech Stack

* HTML5
* jQuery 1.7.2
* Modernizr 2.5.3
* Underscore 1.3.3
* Backbone 0.9.2
